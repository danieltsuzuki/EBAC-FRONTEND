## Desenvolver uma página web de uma agência de design digital chamada "Agência Criativa Web", aplicando conceitos de Design Responsivo, Unidades Relativas, Flexbox e CSS Grid. O site deve ser moderno, atrativo e totalmente adaptável a diferentes tamanhos de tela.

Requisitos:

1. Estrutura HTML:
- Utilizar a estrutura básica de um documento HTML5 (DOCTYPE html, html, head, body).

- Criar um cabeçalho (header) contendo o logo da agência (fictício) e um menu de navegação (nav).

- Criar as seguintes seções (section):

- - Home: Um banner de boas-vindas destacando a agência e seu diferencial.

- - Sobre Nós: Breve descrição da agência e seus valores.

- - Serviços: Lista de serviços oferecidos (ex: design gráfico, desenvolvimento web, branding, etc.).

- - Depoimentos: Comentários de clientes fictícios para reforçar a credibilidade da agência.

- - Contato: Informações para contato (endereço, telefone, e-mail e um formulário de contato).

1. Estilos CSS:
Criar um arquivo CSS externo chamado "estilos.css".

- Aplicar cores, fontes e espaçamentos coerentes com uma identidade visual moderna e profissional.

- Utilizar Flexbox para alinhar itens do menu de navegação e outras seções.

- Utilizar CSS Grid para estruturar a seção "Serviços" em um layout responsivo.

- Usar unidades relativas (%, vh, vw, rem, em, fr) para garantir flexibilidade no design.

- Criar um menu responsivo que se ajuste para telas menores (hambúrguer menu opcional).

- Garantir que imagens e textos sejam responsivos.

3. Funcionalidade e Responsividade:
O site deve ser totalmente responsivo e adaptável para diferentes tamanhos de tela.

- As imagens devem ser otimizadas para responsividade usando srcset.

- O layout deve ser adaptado com media queries (@media) para diferentes resoluções.

- O formulário de contato deve conter campos básicos (input para nome, e-mail e mensagem).
