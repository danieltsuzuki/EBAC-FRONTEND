## Estrutura HTML
Utilizar a estrutura básica de um documento HTML5.

Criar um cabeçalho (header) com o logo da barbearia (imagem fictícia) e um menu de navegação (nav) utilizando as classes do Bootstrap.

### Criar as seguintes seções:

- Home: Banner com imagem de destaque e uma chamada de ação (botão).

- Sobre Nós: Descrição breve da barbearia, destacando seu estilo e proposta.

- Serviços: Cards listando os serviços oferecidos (ex: corte de cabelo, barba, hidratação, etc.).

- Depoimentos: Utilizar o componente Carousel do Bootstrap para exibir comentários de clientes fictícios.

- Contato: Formulário de contato com campos para nome, e-mail e mensagem.

## Estilos CSS com Bootstrap
- Utilizar o sistema de Grid do Bootstrap para estruturar as seções do site.

- Aplicar componentes pré-construídos do Bootstrap (botões, navbar, cards, formulário, carousel).

- Utilizar as classes utilitárias do Bootstrap para ajustes de espaçamento, cores e alinhamentos.

- Personalizar o tema do site com classes utilitárias ou variáveis do Bootstrap (se desejar).

## Responsividade
- O site deve se adaptar perfeitamente a diferentes tamanhos de tela.

- O menu deve ser responsivo, utilizando o componente Navbar do Bootstrap.

- Utilizar o Grid System e media queries do Bootstrap para garantir uma boa experiência em dispositivos móveis.
