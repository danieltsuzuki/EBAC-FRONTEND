# Requisitos
1. Estrutura Inicial

- Crie uma aplicação React (utilize Vite, Create React App ou repositório já existente).

- Estruture um componente de interface chamado CardProduto com as seguintes informações: 

- - Nome do produto

- - Preço

- - Botão "Adicionar ao carrinho"

- Você pode usar dados estáticos para preenchimento.

2. Refatoração com CSS-in-JS

- Utilize Styled Components ou Emotion (escolha uma das duas bibliotecas).

- Transforme os estilos do componente CardProduto para que sejam definidos diretamente dentro do JavaScript utilizando template literals (styled-components ou @emotion/styled).

3. Estilização Dinâmica

- Faça com que o botão mude de cor com base em uma prop chamada adicionado: 

- - true: cor verde (#198754)

- - false: cor cinza (#6c757d)


4. Componentização e Organização

- Mantenha os estilos organizados e separados por componente, mesmo que estejam no mesmo arquivo.

- Use boas práticas de nomeação.
