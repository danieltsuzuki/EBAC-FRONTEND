Neste momento, você criará a sua Primeira Página Web.

## Crie um arquivo HTML e escreva o código necessário para construir uma página web simples com os seguintes elementos:
- Título: "Minha Primeira Página Web" (dentro da tag <title>).

- Cabeçalho: Um título principal (<h1>) com o texto "Bem-vindo ao meu site!".

-Parágrafo: Um parágrafo (<p>) apresentando-se e contando um pouco sobre você.

-Imagem: Insira uma imagem de sua preferência (use a tag <img> e o atributo src para indicar o caminho da imagem).

- Link: Crie um link para um site que você gosta (use a tag <a> e o atributo href).

## Dicas
- Estrutura: Lembre-se de utilizar as tags <html>, <head> e <body> para organizar seu código.

- Formatação: Utilize um editor de código para facilitar a escrita e a visualização do seu código.

- Teste: Abra o arquivo HTML em um navegador para visualizar o resultado.

## Desafio (este item não fará parte dos critérios de avaliação, é apenas para você se desafiar a ir além!)
- Listas: Crie uma lista de suas habilidades ou hobbies utilizando as tags <ul> e <li>.
