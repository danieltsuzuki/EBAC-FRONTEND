# Objetivo do exercício:

Criar uma aplicação de cadastro de usuários que utilize Fetch API para preencher automaticamente o endereço a partir do CEP (usando a API do ViaCEP) e que salve os dados do formulário no Web Storage API, garantindo que as informações permaneçam preenchidas mesmo após o usuário recarregar a página.

## Requisitos:

### Estrutura do Projeto:

- Criar um projeto com os seguintes arquivos:

- index.html (Interface da aplicação)

- scripts.js (Código JavaScript para interagir com a API)

- styles.css (Estilos básicos, opcional)

## Funcionalidades da Aplicação:

- Preenchimento automático do endereço ao digitar o CEP (usando Fetch API e ViaCEP).

- Salvar os dados preenchidos no Web Storage API para evitar que o usuário perca as informações ao recarregar a página.

- Restaurar os dados do formulário ao carregar a página.
